1. Diferencias entre etiquetas similares

<b> vs <strong>

<b> solo pone el texto en negritas. <strong> también pone el texto en negritas, pero semánticamente indica importancia para motores de búsqueda como Google.

<i> vs <em>

<i> se usa para texto en cursiva. <em> también da cursiva, pero, igual que con <strong>, semanticamente indica importancia, lo que es útil para motores de búsqueda como Google.

<s> vs <del>

<s> muestra el texto tachado. <del> también lo tacha, pero semánticamente indica que el texto fue eliminado o ya no es válido, lo que es útil para motores de búsqueda como Google.

2. Elementos en línea y de bloque

Los elementos en línea se colocan uno al lado del otro y no generan un salto de línea. Los de bloque ocupan todo el ancho y comienzan en una nueva línea.

Ejemplos de elementos en línea:

  • <span>: Agrupa o marca el contenido sin alterar la estructura.
  • <a>: Crea enlaces a otras ligas web.
  • <img>: Inserta imagenes a nuestra página.
  • <strong>: Las negritas son elementos en línea.
  • <i>: Las cursivas también son elementos en línea

Ejemplos de elementos de bloque:

  • <div>: Divisiones.
  • <p>: Parrafos.
  • <h1> a <h6>: Encabezados o títulos.
  • <ul> y <ol>: Listas.
  • <table>: Tablas.
